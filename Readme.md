Bite Bistro is a responsive, single-page landing website designed for a modern fast-food restaurant. The design focuses on high-quality visuals ("Gourmet Fast Food"), clear typography, and a user-friendly mobile experience.

ðŸš€ Features

1. Navigation Bar (Responsive)

Sticky Top: Remains visible while scrolling.

Responsive Behavior:

Desktop: Displays full text links (Home, Menu, About, Contact).

Tablets/Mobile (< 1200px): Collapses into a "Hamburger" menu to prevent text overlap.

Visuals: Dark theme (bg-dark) with Gold accents (text-warning).

2. Hero Section

Background: Full-screen (100vh) parallax image with a dark linear gradient overlay to ensure text readability.

Call to Action: Centered "View Menu" button with hover elevation effects.

3. Menu Section

Grid Layout: Uses Bootstrap's grid system (col-md-6) to display food items.

Cards: Custom styled cards featuring high-quality food photography, descriptions, price tags, and "Munch Now" buttons.

4. About & Contact

About: Split-screen layout with text on one side and a featured interior shot on the other.

Contact: Functional-looking form styled with Bootstrap form controls.

Technologies Used

HTML5: Semantic structure (header, nav, section, footer).

CSS3: Custom styling for the Hero section overlay, button animations, and font weights.

Bootstrap 5.3: Core framework for the grid system, responsive utilities, and pre-built components (Navbar, Cards, Forms).

Bootstrap Icons (v1.11.3): Used for social media icons (Instagram) in the footer.

Google Fonts:

Lato: Headings and Brand Name.

Roboto: Body text and paragraphs.
